How Binary Search Works

Binary search works on sorted arrays by repeatedly halving the search space:

  1. Find the middle element of the current range
  2. If middle equals target → found!
  3. If target is less than middle → search left half
  4. If target is greater than middle → search right half
  5. Repeat until found or range is empty

Time Complexity: O(log n) — much faster than linear search O(n)
