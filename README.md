# Day 22 — Binary Search Visualizer

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
An interactive binary search visualizer that shows
how binary search works step by step.

## How to view
Open `index.html` in your browser.

## Features
- Step by step binary search animation
- Visual left, mid and right pointers
- Custom array and target input
- Comparison counter
- Algorithm explanation
